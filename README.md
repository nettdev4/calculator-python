# Калькулятор на Python

Простой консольный калькулятор, написанный на Python.

## Возможности

- Сложение чисел
- Вычитание чисел
- Умножение чисел
- Деление чисел
- Проверка ошибок ввода
- Защита от деления на ноль

## Запуск

Скачайте файл `calculator.py` и выполните команду:

```bash
python calculator.py
